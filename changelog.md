# 幻昼更新日志

## 1.2 公测版本

本次更新重点完成了实验真流式 Agent、记忆称呼优化、主动消息通知修复和部分 UI 细节调整。

1. 实验真流式 Agent 已完成
新增“实验真流式输出”线路，默认仍保持原稳定线路。开启后，普通回复和带工具调用的 Agent 回复都可以走真流式流程，模型输出会实时进入聊天气泡。

同时支持：
- 文本增量实时显示
- 工具调用参数流式累积
- 工具执行状态显示
- 工具执行后继续模型续写
- 工具结果缓存命中状态输出
- 停止按钮取消当前 SSE / Agent 任务
- 真流式不可用时自动回退稳定线路

2. 稳定性保护
真流式是实验线路，不会影响默认聊天。只有在聊天设置中手动开启后才启用。
如果真流式在工具执行前失败，会自动切回稳定线路；如果工具已经执行，则不会自动重跑，避免重复执行副作用工具。

3. 流式 UI 修复
修复了顶部“正在输入中”区域误显示回复正文的问题。
现在：
- 顶部只显示状态，例如“正在输入中”“正在使用工具”
- 聊天列表底部显示真正的流式草稿气泡
- 流式过程中会过滤隐藏记忆块，避免 <huanzhou-memory> JSON 短暂露出

4. 记忆称呼优化
修复结构化记忆容易写成“用户自称叫 XXX”的问题。
现在记忆抽取会注入当前用户显示名，并要求模型不要用泛称“用户”作为主语。
同时增加兜底修正和旧数据显示兼容，让记忆页展示更自然。

5. 后台主动消息显示与通知修复
主动消息现在会被标记为 proactive。
聊天列表会显示：
“主动消息 · 内容预览”

聊天气泡中也会显示“主动消息”标签。

同时修复了“系统通知权限已给但收不到主动消息通知”的问题。原因是应用内还有一个“新消息通知”开关，默认可能未开启。现在：
- 开启后台主动消息时，如果通知权限已允许，会自动开启并保存新消息通知
- 授权通知成功后，也会自动保存新消息通知开关
- 新消息通知开关切换后立即保存
- 通知发送失败会写日志，便于判断是权限、渠道关闭还是其他原因

6. 小记页面 UI 调整
优化了小记页顶部“今日心情”卡片的排版，增加内部边距，调整右侧心情数值对齐。
